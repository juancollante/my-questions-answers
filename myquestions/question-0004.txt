Problema de Machine Learning (Prediccion de Aprobacion de Prestamo)

Un banco desea predecir si una solicitud de prestamo sera aprobada o no a partir de datos historicos de clientes.

Se dispone de un archivo CSV que contiene informacion de multiples solicitudes con las siguientes columnas:

ingreso_mensual
edad
score_crediticio
deudas
monto_solicitado
aprobado (variable objetivo: 0 = no aprobado, 1 = aprobado)

Problema

Debes desarrollar una funcion en Python que:

1. Cargue el dataset desde un archivo CSV
2. Limpie los datos (manejo de valores nulos)
3. Separe variables independientes (X) y variable objetivo (y)
4. Divida los datos en entrenamiento y prueba
5. Entrene un modelo de clasificacion usando LogisticRegression
6. Utilice el modelo entrenado para predecir la aprobacion del prestamo en el conjunto de prueba
7. Evalue el modelo usando accuracy

Especificacion exacta de la funcion

Debes implementar la siguiente funcion:

def predecir_aprobacion_prestamo(ruta_csv: str, test_size: float, random_state: int) -> tuple:

Argumentos

ruta_csv (str): Ruta del archivo CSV con los datos de las solicitudes de prestamo
test_size (float): Proporcion del dataset que se usara como conjunto de prueba (por ejemplo, 0.2)
random_state (int): Semilla para asegurar reproducibilidad en la separacion de datos y en el modelo

Requisitos tecnicos

Debes usar obligatoriamente:

pandas para cargar y manipular los datos
numpy para el manejo numerico
sklearn:
train_test_split
LogisticRegression
accuracy_score

Salida esperada

La funcion debe retornar:

(predicciones, accuracy)

Donde:

predicciones: un arreglo (numpy array o lista) con las predicciones del modelo para todas las solicitudes del conjunto de prueba (valores 0 o 1)
accuracy: un numero flotante entre 0 y 1 que representa la exactitud del modelo

Consideraciones importantes

No debes incluir la columna aprobado dentro de X
Debes entrenar el modelo antes de hacer predicciones
Las predicciones deben corresponder unicamente al conjunto de prueba
Debes manejar valores nulos (puedes eliminarlos o imputarlos)
El flujo debe ser reproducible usando random_state

Ejemplo de uso esperado

preds, acc = predecir_aprobacion_prestamo("solicitudes.csv", 0.2, 42)

print("Predicciones:", preds)
print("Accuracy:", acc)

Enfoque conceptual

Este problema evalua la comprension del flujo completo de machine learning para clasificacion binaria: preparacion de datos, entrenamiento supervisado, prediccion sobre nuevos casos y evaluacion del desempeno del modelo.
