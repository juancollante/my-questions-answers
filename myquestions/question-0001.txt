Problema de Machine Learning (Prediccion de Falla de Maquinaria)

Una planta industrial desea predecir si una maquina presentara falla en el proximo ciclo de operacion, usando datos historicos de sensores.

Se dispone de un archivo CSV que contiene informacion de multiples registros con las siguientes columnas:

temperatura_motor
vibracion
horas_uso
presion
ciclos_mantenimiento
falla (variable objetivo: 0 = no falla, 1 = falla)

Problema

Debes desarrollar una funcion en Python que:

1. Cargue el dataset desde un archivo CSV
2. Limpie los datos (manejo de valores nulos)
3. Separe variables independientes (X) y variable objetivo (y)
4. Divida los datos en entrenamiento y prueba
5. Entrene un modelo de clasificacion usando KNeighborsClassifier
6. Utilice el modelo entrenado para predecir fallas en el conjunto de prueba
7. Evalue el modelo usando accuracy

Especificacion exacta de la funcion

Debes implementar la siguiente funcion:

def predecir_falla_maquinaria(ruta_csv: str, test_size: float, n_neighbors: int) -> tuple:

Argumentos

ruta_csv (str): Ruta del archivo CSV con los datos de sensores
test_size (float): Proporcion del dataset que se usara como conjunto de prueba (por ejemplo, 0.2)
n_neighbors (int): Numero de vecinos a usar en KNeighborsClassifier

Requisitos tecnicos

Debes usar obligatoriamente:

pandas para cargar y manipular los datos
numpy para el manejo numerico
sklearn:
train_test_split
KNeighborsClassifier
accuracy_score

Salida esperada

La funcion debe retornar:

(predicciones, accuracy)

Donde:

predicciones: un arreglo (numpy array o lista) con las predicciones del modelo para todos los registros del conjunto de prueba (valores 0 o 1)
accuracy: un numero flotante entre 0 y 1 que representa la exactitud del modelo

Consideraciones importantes

No debes incluir la columna falla dentro de X
Debes entrenar el modelo antes de hacer predicciones
Las predicciones deben corresponder unicamente al conjunto de prueba
Debes manejar valores nulos (puedes eliminarlos o imputarlos)
Si n_neighbors es mayor que el numero de muestras de entrenamiento, debes ajustarlo al maximo valido

Ejemplo de uso esperado

preds, acc = predecir_falla_maquinaria("maquinas.csv", 0.2, 5)

print("Predicciones:", preds)
print("Accuracy:", acc)

Enfoque conceptual

Este problema evalua la comprension del flujo completo de machine learning para clasificacion binaria en contexto industrial: limpieza, entrenamiento, prediccion y evaluacion del desempeno.
